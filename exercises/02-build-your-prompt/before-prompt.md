# Before: Naive Prompt

This is a typical prompt that most people would write. It works, but it lacks context.

---

## The Prompt

```
Help me write a summary of my project status for my team meeting.
```

---

## What Is Missing?

Think about these questions:

- Who is the team? Engineers? Executives? Mixed?
- What tone is appropriate? Formal? Casual? Urgent?
- What should be included? Just milestones? Risks? Blockers?
- What should be excluded? Budget details? Personnel issues?
- How long should it be? Two sentences? A full page?
- What happens after this summary is delivered? Decisions? Action items?

Without context, the AI has to guess all of these. And it will guess wrong.
