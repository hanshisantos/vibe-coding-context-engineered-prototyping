# Participant Setup

Complete these steps **before** the workshop begins.

---

## What You Need

### Required
- A laptop or tablet with a web browser
- Access to this repository: [github.com/hanshisantos/vibe-coding-context-engineered-prototyping](https://github.com/hanshisantos/vibe-coding-context-engineered-prototyping)
- A text editor (VS Code, Notepad, Google Docs, or even pen and paper)

### Recommended (but not required)
- Access to an AI tool for testing prompts. Any of these work:
  - GitHub Copilot
  - Microsoft Copilot (copilot.microsoft.com)
  - ChatGPT (chatgpt.com)
  - Claude (claude.ai)
  - Google Gemini (gemini.google.com)
- A free account on any of the above is sufficient

### If You Cannot Access an AI Tool
That is fine. The exercises are designed to work without one. You will build prompts on paper and peer-review with a neighbor. The learning is in the construction, not the tool.

---

## Before the Session

1. **Open the repository** in your browser and bookmark it
2. **Download or clone the repo** if you want local copies of the exercises:
   ```
   git clone https://github.com/hanshisantos/vibe-coding-context-engineered-prototyping.git
   ```
3. **Test your AI tool access** — send any simple prompt and confirm you get a response
4. **Think about a task** where you need AI help this week. You will use it in Exercise 1.

---

## Repository Structure

Here is what you will find in the repo:

```
slides/                  ← Workshop slide deck (Marp markdown)
exercises/
  01-define-your-vibe/   ← Exercise 1: VIBE worksheet
  02-build-your-prompt/  ← Exercise 2: Before/after prompt building
  03-rapid-prototype/    ← Exercise 3: Take-home prototype template
examples/                ← 3 fully worked reference examples
resources/               ← Cheat sheet, prompt patterns, AI checklist
instructor-guide/        ← Facilitator notes (for trainers)
```

During the workshop, the trainer will tell you which folder to open at each step.

---

## FAQ

**Do I need to know how to code?**
No. This workshop is about designing AI interactions, not writing software.

**Do I need a paid AI subscription?**
No. Free tiers on any AI tool are sufficient. You can also complete the exercises without an AI tool.

**Can I use this material after the workshop?**
Yes. The repository is public. Use it, share it, build on it.

**What if I get stuck during an exercise?**
Raise your hand or ask a neighbor. The trainer will be walking the room.
